# 📖 [Recipe-Book-Website] <a name="about-project"></a>

> **[Recipe-Book-Website]** Developed a website where user can share and organize favorite recipes. It Includes functionality for adding, displaying, and deleting recipes. This web app uses HTML for structure, CSS for styling, and JavaScript for interaction.
